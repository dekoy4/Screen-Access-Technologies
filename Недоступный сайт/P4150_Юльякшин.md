**Тематика сайта**  

*Шуточный сайт, на котором написано "Нажмите на кнопку, чтобы получить приз!" (Кнопка не нажимается)*

**Структура сайта**

Сайт представляет собой одиночный HTML-файл с минимальной статической разметкой, где весь видимый контент генерируется динамически через JavaScript и Canvas. Основа — стандартный HTML5-скелет без семантических тегов.
1. *<head>: Содержит meta charset="UTF-8", пустой <title>, и блок <style> с правилами CSS для скрытия текста (font-size: 0), отключения outline, абсолютного позиционирования и стилей для canvas.*
2. *<body>: Только два элемента:*
 2.1 *Скрытый <div class="hidden" role="none"> — содержит фейковый текст, positioned off-screen (left: -9999px).*
 2.2 *<canvas id="canvas" tabindex="-1" aria-hidden="true"> — основной контейнер для графического рендеринга.*

JavaScript-логика:
1. Скрипт в <script> активируется на событии DOMContentLoaded и управляет всей динамикой:
2. Получает контекст Canvas и устанавливает размеры под окно браузера.
3. Рисует текст (fillText) и элементы интерфейса (кнопка как прямоугольник) напрямую на холсте.
4. Блокирует клавиатурные события (keydown preventDefault()).
5. Удаляет все ARIA-атрибуты и роли с элементов, устанавливая tabIndex = -1.

CSS-слои:
Стили применяются глобально (* { ... }):
1. *body: overflow: hidden, font-size: 0 — предотвращает чтение любого статического текста.*
2. *.hidden: Техника clip для screen readers.*
3. *#canvas: Полноэкранный (100vw/100vh).*


Эта структура обеспечивает нулевую семантику: нет заголовков, кнопок или ссылок — только "немой" canvas, который screen readers интерпретируют как пустой график.
